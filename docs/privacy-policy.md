# プライバシーポリシー (Privacy Policy)

**最終更新日: 2025年12月1日**

Jifree（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、本拡張機能が収集するデータとその利用方法について説明します。

## 1. 収集するデータとその目的

本拡張機能は、以下のデータを収集・使用します。

### 1.1 ウェブサイトのコンテンツ（テキスト）
- **収集するデータ**: ユーザーがWebページ上でドラッグして選択したテキストデータ。
- **利用目的**: 選択されたテキストに含まれる漢字にふりがな（ルビ）を振るため。
- **処理方法**: テキストデータはHTTPS通信を通じて当社のバックエンドサーバーに送信され、形態素分析が行われます。解析結果（ふりがなデータ）を返却した後、サーバー上でテキストデータは破棄され、永続的に保存されることはありません。

### 1.2 ユーザーのアクティビティ（操作ログ）
- **収集するデータ**: テキスト選択（マウス操作）のイベント情報。
- **利用目的**: ユーザーがテキストを選択したタイミングを検知し、ポップアップを表示するため。
- **処理方法**: この処理はすべてユーザーのブラウザ内（ローカル）で行われ、外部サーバーへ送信されることはありません。

### 1.3 匿名識別子 (User ID)
- **収集するデータ**: ランダムに生成されたUUID。
- **利用目的**: APIの過剰な利用を防ぐためのレート制限（Rate Limiting）を行うため。
- **処理方法**: 初回起動時に生成され、ブラウザのローカルストレージ (`chrome.storage.local`) に保存されます。個人を特定する情報（名前、メールアドレス等）とは一切紐付きません。

## 2. データの共有と第三者への提供

本拡張機能は、収集したデータを第三者に販売、貸与、または共有することはありません。ただし、法令に基づき開示が必要な場合を除きます。

## 3. データの保存期間

- **テキストデータ**: 解析処理が完了した時点で即座にメモリから削除されます。
- **アクセスログ**: サーバーの運用管理およびセキュリティ維持のため、一時的なアクセスログ（IPアドレス、タイムスタンプ等）がGoogle Cloud Platform上に一定期間保存される場合がありますが、これらは自動的に削除されます。
