# Jifree (ジフリー) - 漢字読み方支援 Chrome拡張機能

**【漢字が読めないお子様でも、インターネットの世界を自由に楽しめます！】**

Jifree（ジフリー）は、Webページ上の漢字に簡単に「ふりがな（ルビ）」を振ることができるChrome拡張機能です。
難しい漢字が読めなくて困っているお子様や、日本語を学習中の方に最適です。

## ✨ 主な機能とメリット

1. **かんたん操作**:
   読みたい文章をマウスでなぞる（ドラッグする）だけ！自動的に漢字の上にふりがなが表示されます。
2. **邪魔にならない**:
   ページ全体のレイアウトを崩さず、選択した部分にだけふりがなを表示するので、快適に読み進められます。
3. **高い精度**:
   文脈に合わせて正しい読み方を判断するので、「辛い（からい/つらい）」のような読み分けもバッチリです。
4. **安心・安全**:
   お子様が使うことを想定し、余計な広告やデータ収集は一切ありません。

## 🚀 こんな方におすすめ

* まだ習っていない漢字がある小学生のお子様
* Webサイトで調べ学習をしたいけれど、漢字が難しくて読めない方
* 日本語の読み方を勉強中の留学生の方
